# SITE CONSTITUTION: a7 Arquitectura Portfolio

## 1. Vision & Purpose
A high-end, minimalist portfolio for "a7 Arquitectura", a studio based in Buenos Aires. The site focuses on visual impact, architectural photography, and a clean, professional user experience.

## 2. Technical Stack
- **Framework:** React 19 (Vite)
- **Routing:** React Router 7
- **Styling:** Vanilla CSS with CSS Variables
- **Architecture:** Component-based, centralized data in `src/data/projects.js`

## 3. Sitemap
- [x] Home (`/`) - Hero + Vision statement
- [x] About (`/nosotros`) - Studio history and team
- [x] Projects (`/proyectos`) - Masonry gallery of work
- [x] Project Detail (`/proyectos/:id`) - In-depth project view + Lightbox
- [x] Contact (`/contactanos`) - Lead generation form
- [x] 404 (`*`) - Catch-all error page

## 5. Design System (Summary)

- **Typography:** Inter / Sans-serif stack

- **Vibe:** Minimalist, Grid-based, Grayscale/Low-saturation imagery
